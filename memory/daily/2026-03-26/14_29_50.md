### 第一步：启动后端服务（打开第一个终端）
1. 在你的键盘上按 `Win + R`，输入 `cmd`，然后按回车，弹出一个黑框框。
2. 在黑框框里，依次敲入以下命令（每敲完一行按一次回车）：
   ```cmd
   cd /d C:\AI\second-brain-ai\backend
   python -m uvicorn main:app --host 127.0.0.1 --port 8000
   ```
3. 等它屏幕上出现 `Application startup complete` 的字样，说明后端跑起来了。**把这个黑框框放一边，不要关掉它。**

---

### 第二步：启动前端服务（打开第二个终端）
1. 再按一次 `Win + R`，输入 `cmd`，回车，弹出**第二个**全新的黑框框。
2. 在这个新黑框框里，依次敲入以下命令（每敲完一行按回车）：
   ```cmd
   cd /d C:\AI\second-brain-ai\frontend
   npm run dev
   ```
3. 等几秒钟，你会看到屏幕上出现绿色的字，类似：
   `➜  Local:   http://localhost:5173/`

---

### 第三步：去浏览器抓虫子 🐛
1. 打开你电脑上的浏览器（推荐 Chrome 或 Edge）。
2. 在网址栏输入：`http://localhost:5173`，然后敲回车。
3. 这时候应该会显示那个“正在初始化中”的卡死页面。
4. **按键盘上的 `F12` 键**（或者在网页空白处 右键 -> 检查）。
5. 屏幕右边或下面会弹出一堆复杂的面板。在最上面一排的标签里，找到 **Console（控制台）** 并点它。
