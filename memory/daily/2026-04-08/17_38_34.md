这句报错的意思翻译成人话就是：“**你不能在自己的房间里穿衣服的时候，去隔壁房间拉电闸！**”
React 的铁律是：一个组件在渲染（画自己）的时候，绝对不能去改变其他组件（尤其是它的父组件大心脏 `NovaBlockEditor`）的状态。

在绑定点击事件的时候，小弟把**“点击时才执行的动作”**（`onClick={() => onSelect(sticker)}`）
手滑写成了**“一画出来就立刻执行的动作”**（`onClick={onSelect(sticker)}`）！
结果就是：贴纸一画出来，它就“啪”地一下直接去戳大心脏（调用了父组件的 `onSelect` 或 `onUpdate`），大心脏受了刺激马上全页刷新，贴纸又被重新画一遍，又戳一下……直接变成死亡螺旋！
