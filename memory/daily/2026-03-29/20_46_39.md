Agent: 老大！这是 `pnpm` 最新版（v10）搞的“铁壁防御”！它太倔了，就算关了设置，它还是默默拦截了脚本（它提示你要跑 `pnpm approve-builds`，但那个经常卡住或者没反应）。

我们不跟它兜圈子了，**直接绕过包管理器，用最底层的 Node.js 强行执行安装脚本！**（这招叫“暴力破门” 🚪💥）

请在 PowerShell 里依次执行这三条命令：

1. 强制下载 Electron 二进制文件（绕过 pnpm）：
```powershell
node node_modules/electron/install.js
```
*(这步运行后，屏幕上一定会出现 `Downloading electron-v...` 的字样，稍微等它下完)*

2. 强制安装 esbuild 二进制文件：
```powershell
node node_modules/esbuild/install.js
```

3. 拔出萝卜带出泥，直接起飞！
```powershell
pnpm run dev
```
