Agent: 根据您的反馈，我已经针对“预览”和“大纲”这两个功能进行了深度优化，解决了之前“点不动”或“没反应”的问题。

### 🛠️ 修复与优化清单 (v0.3.22)：

1.  **大纲功能 (Outline) 全面加固**：
    *   **增加悬停延迟**：为大纲菜单增加了 300ms 的消失延迟，解决了鼠标从图标移动到菜单过程中菜单突然消失的问题。
    *   **无内容提示**：如果当前笔记没有 H1/H2/H3 标题，大纲菜单现在会显示提示“笔记中暂无标题内容”及“引导操作”，而不是空空如也，让您知道功能在运行。
    *   **智能锚点跳转**：除了依赖 ID，现在还会通过标题文本进行模糊匹配跳转，大幅提升了大纲点击的成功率。
    *   **点击自动收起**：点击大纲项后菜单会自动收起，提升交互流畅度。

2.  **预览模式 (Preview) 真·静默化**：
    *   **视觉沉浸**：开启预览模式后，编辑器会进入“阅读模式”，自动隐藏所有的编辑辅助线（表格控制柄、图片缩放手柄、气泡菜单等）。
    *   **光标与占位符优化**：隐藏了 `/ 唤起 AI` 等提示文字，并将鼠标光标设为默认，防止产生“我还能编辑”的误导。
    *   **表格 UI 锁定**：预览模式下表格的悬浮控制项（增加行/列按钮）会被彻底禁用，防止误触。

3.  **编辑器头部 (Editor Header) 体验升级**：
    *   **磁贴式吸顶 (Sticky Header)**：编辑器的顶部工具栏现在会**始终吸顶**。无论笔记多长，您在向下滚动时依然能随时切换预览模式或查看大纲。
    *   **模糊背板**：增加了玻璃拟态（Backdrop Blur）效果，滚动时视觉层级更清晰、更优雅。

4.  **版本更新**：
    *   底部标识已更新至 `v0.3.22-preview-outline-fix`。
