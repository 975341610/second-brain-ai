### ⚙️ 当前状态

用户只需在 Windows 机器上：
1. 安装好 **Python 3.11+** 和 **Node.js LTS**
2. 双击运行 **`build_windows.bat`**
3. 等待约 5~10 分钟构建完成，即可得到一个 `Setup.exe` 安装包和便携版 `SecondBrainAI.exe`
4. 安装后从"开始菜单"或桌面快捷方式启动，浏览器会**自动弹出**
