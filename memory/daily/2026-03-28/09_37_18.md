### 1. 静态资源路径 (Assets & Icons)
咱们之前比如：

### 2. Python 后端拉起路径 (Sidecar)
这绝对是最核心的雷区！
咱们现在的 Electron 壳子启动时，需要静默拉起咱们的 Python 后端（`uvicorn main:app`）。之前咱们把路径写成了类似 `../backend/...`，现在 Electron 主进程的代码被移动到了 `src/main` 里，打包后又在 `out/main`，如果这时候去相对找 `backend` 目录，大概率会扑空，导致“界面出来了，但一直在转圈连不上数据库”。

### 3. 前端 Tailwind CSS 的扫描范围
`tailwind.config.js` 里有个 `content` 字段，专门告诉系统去哪些文件夹里扫描 CSS class。之前写的是 `frontend/src/**/*.{js,ts,jsx,tsx}`，如果重构时没有精准改成 `src/renderer/src/**/*.{js,ts,jsx,tsx}`，咱们之前精心调的那套“水晶多巴胺”透明质感，可能就会因为 CSS 没打包进去而一夜回到解放前（变回白底黑字）。
