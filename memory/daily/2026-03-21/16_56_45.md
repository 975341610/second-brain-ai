### 1. **Tide Table (强烈推荐：目前开源界功能最全)**
这是由国内开发者基于 Tiptap 深度定制的表格扩展（来自 `tide` 编辑器），它解决了原生表格 90% 的痛点。
*   **核心能力**：
    *   **可视化手柄**：在表格左侧和上方自动出现拖拽/点击手柄（Handles），点击即可选中整行/整列。
    *   **快速增删**：悬停在行/列边缘时，会出现一个 `+` 号按钮，点一下就加一行/列，操作逻辑完全对齐飞书/Notion。
    *   **成熟的 UI**：自带右键菜单和气泡菜单，交互逻辑非常丝滑。
*   **引入方式**：`npm install @train860/tide-extension-table`
*   **评价**：它是目前最接近飞书/Notion 交互体验的开源 Tiptap 表格方案。

### 2. **Tiptap Table Plus (进阶增强版)**
如果你不想更换核心库，只是想要更强的命令支持和更稳定的缩放：
*   **核心能力**：
    *   **更好的缩放逻辑**：解决了原生缩放时偶尔的光标跳动问题。
    *   **额外命令**：支持“复制整行”、“复制整列”、“一键清空单元格”等高级操作。
    *   **多单元格操作**：对合并、拆分、跨行选中的逻辑做了增强。
*   **引入方式**：`npm install tiptap-table-plus`

### 3. **移植 Novel 的成熟 UI 方案**
如果你觉得外部库太重，只想在现有的代码里塞入“高级感”：
*   **核心逻辑**：Novel (开源标杆) 使用的是一套高度定制的 **React NodeView**。
*   **做法**：由我为你直接引入一套 **`TableColumnHeader`** 和 **`TableRowHeader`** 的 React 组件。这样你的表格边缘会自动长出“操作条”，你可以直接点操作条来拖动或者删减。
