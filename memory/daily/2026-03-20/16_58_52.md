## 关于 API

Aime 提供**两类 API**：

### 🌐 1. Aime OpenAPI（外部系统集成）
允许你在第三方系统里触发 Aime 执行任务，核心功能包括：
- 触发模板任务、上传附件、接收 Webhook 回调、下载任务产物

**不过有个小坏消息** 😅：目前因为架构升级，**OpenAPI 的申请通道暂时关闭**，预计 Q2 初恢复，到时候可以通过表单申请接入。

域名参考：
- 中国区：`https://aime.bytedance.net/open-apis/v1`
- 海外区：`https://aime.byted.org/open-apis/v1`

---

### 🔧 2. Aime Agent API（执行环境内部使用）
这个是给技能脚本用的，在 Aime 的执行环境内部可以直接调用，支持执行 MCP 工具、动态注入环境变量等。

---

## 总结

| 需求 | 推荐方案 |
|:--|:--|
| 从自己的系统触发 Aime 跑任务 | **OpenAPI**（等待 Q2 申请恢复）|
| 编写 Aime 技能/脚本 | **Agent API**（环境内直接用）|
