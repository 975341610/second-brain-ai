Agent: 推送成功！v0.4.3 已上线 ✨

**这次修了两个 bug：**

**🔧 自动保存乱跳 + 鼠标失踪**
- 根因：同步内容的 `useEffect` 里有个 `!editor.isFocused` 判断，有时 blur 延迟导致误触发 `setContent`，强行重建了编辑器 DOM，DragHandle 就跟丢了鼠标
- 修复：移除了这个不稳定的条件判断，改为**只要 note.id 真正变化才 setContent**，同一篇笔记的任何保存回流都绝对不会碰编辑器内容

**🔧 git 命令找不到**
- 根因：PyInstaller 打包的 EXE 启动时 PATH 是残缺的，`shutil.which("git")` 找不到
- 修复：现在会主动探测 Windows 上 Git 的常见安装路径（`C:\Program Files\Git\...`），找到了就直接用绝对路径调用，不再依赖 PATH
