**🐛 Bug 1：自动保存 + 鼠标不跟随**
根因在 `NotionEditor.tsx` 第 391 行这个条件：
```ts
if (!editor.isFocused || lastSyncedNoteIdRef.current === undefined) {
```
逻辑是**"或"**——只要 `lastSyncedNoteIdRef.current === undefined`（首次加载）就会触发 `setContent`，而且 `isFocused` 是 Tiptap 内部状态，切换笔记时有时候 blur 有延迟，导致条件判断失误，强行 `setContent` 重置了整个编辑器 DOM → DragHandle 丢失鼠标追踪。

修复方案：改为只在 **note.id 真正变化** 时才 setContent，完全不依赖 `isFocused` 判断。

---

**🐛 Bug 2：git 命令找不到**
`shutil.which("git")` 在 Windows EXE 打包环境里依赖系统 PATH，但 PyInstaller 启动时 PATH 是残缺的（没有 `C:\Program Files\Git\cmd`）。

修复方案：主动探测 Git 的常见安装路径，找到就直接用绝对路径调用，不再依赖 PATH。
