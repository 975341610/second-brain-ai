咱们的项目是一个前后端分离的结构，根目录的 `package.json` 主要管 Electron，而**真正需要这个 `@tiptap/extension-collaboration` 插件的是前端文件夹里的 `package.json`！**

所以根目录的 `npm install` 没有把它装到前端的模块里去。

解决办法很简单，咱们进到前端目录去装一下就好了！请在终端依次执行下面这三步：

1. 先按 `Ctrl + C` 停掉当前的开发服务器
2. **进入前端目录**并安装依赖：
   ```bash
   cd frontend
   npm install
   ```
3. 装完后**退回到根目录**重新启动：
   ```bash
   cd ..
   npm run dev
   ```
