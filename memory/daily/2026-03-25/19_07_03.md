### 🚀 3 分钟 Tailscale 极简部署指南

**第一步：在你的“主电脑”（跑笔记软件的 Windows）上安装**
1. 打开浏览器，访问官网：[tailscale.com](https://tailscale.com)
2. 点击右上角 **Try for free**，用你的微软/Google/GitHub 账号随便授权登录一下（这是为了创建你私人的虚拟网络）。
3. 登录后，下载 Windows 客户端并无脑下一步安装。
4. 安装完成后，右下角状态栏会出现一个 Tailscale 的小图标，右键点击它，选择 **Log in**，用刚才的账号登录。
5. ⚠️ **关键点**：鼠标悬停在右下角 Tailscale 的图标上，会显示一个以 `100.` 开头的 IP 地址（比如 `100.123.45.67`）。**把这个 IP 记下来！**这就是你的专属神仙域名了。

**第二步：配置主电脑的防火墙（放行笔记软件）**
为了让手机能连进来，需要让 Windows 防火墙不拦着它：
1. 在电脑上运行你的 `SecondBrainAI.exe`，让它保持开启。
2. 按 `Win` 键，搜索“防火墙”，打开 **“允许应用通过 Windows 防火墙”**。
3. 找到你的 `SecondBrainAI.exe`（或者相关的 Python/uvicorn 进程），把后面的**“专用”**和**“公用”**都打上勾，点击确定。

**第三步：在你的“手机/外地电脑”上访问**
1. 你的手机（iPhone/安卓），直接去应用商店下载 **Tailscale** APP。
2. 打开 APP，登录你**刚才同一个账号**。
3. 手机连接成功后，状态栏会出现 VPN 图标（不用担心，它不影响你上正常的网，只是多开了一条隧道）。
4. 在手机浏览器里，输入刚才记下来的 IP 加上端口号，比如：
   👉 **`http://100.123.45.67:8765`**
   （注意！如果是用刚才更新的代码，这时候会弹出一个**极简的密码登录页**，输入你在设置里配的 `Access Token`，就能直接进到主界面！）
