不过别担心，最彻底、最干净的办法是借用它里面自带的命令行戳一下，只需两步，非常简单：

**👉 第一步：打开自带终端**
打开 GitHub Desktop，在最顶部的菜单栏，点击 **Repository** -> 选择 **Open in Command Prompt**（有些电脑上可能叫 Open in Terminal / Open in Git Bash，快捷键是 `Ctrl + ~` 或 `Ctrl + \`）。

**👉 第二步：输入回滚命令**
弹出一个黑框框后，把你本地当前的修改（如果有）都清空，然后直接把这行命令复制粘贴进去，敲回车：
```bash
git reset --hard ea215b6b004eb0d640993c3942dc721bd0cc6073
```
*(执行完这句，你本地的代码瞬间就会变回 `fix: 修复 TableOfContents...` 这个干干净净的旧版本啦！)*
