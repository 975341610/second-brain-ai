# Nova Project v0.17.0 更新说明

**发布日期**：2026-04-15  
**版本号**：0.17.0 (Stable)

## 🚀 主要更新点

本次升级标志着 Nova 进入 **0.17 阶段**，核心聚焦于 **AI 写作体验的极致优化**、**编辑器基础架构的稳定性** 以及 **本地离线优先功能的深度集成**。

1. **AI 体验 Pro Max**：引入了行内像素女仆 (Inline Pixel Maid) 加载动画，彻底重构了流式文本的实时反馈机制。
2. **编辑器基础修复**：彻底解决了拖拽指示线残留、标题折叠三角显示、Slash 菜单闪退等长期遗留的 UI/UX 问题。
3. **GoodNotes 风格背景**：实现了高性能 CSS 背景纸功能，支持点阵、线稿、格子纹理，提升手写与排版沉浸感。
4. **性能深度优化**：通过侧边栏与编辑器的绝对定位解耦，实现了在高负载文档下的 60 FPS 稳定表现。
5. **本地模型支持**：初步搭建了 `llama-cpp-python` 的本地 AI 引擎脚手架，支持硬件检测与上下文长度动态调节。

---

## 🛠️ 已包含功能清单

### 🖋️ 编辑器 (Editor)
- **高级大纲 (TOC)**：极简主义 2.5D 大纲，支持滚动实时同步与毛玻璃展开效果。
- **自动序号注脚 (Footnote)**：支持全自动序号重排、双击编辑与悬停 Tooltip 预览。
- **2.5D Coverflow 画廊**：图片 Slider 升级为 2.5D 景深画廊，支持滚轮切图与物理弹簧动画。
- **莫兰迪双链 (NoteLink)**：重构了双向链接 UI，采用 React NodeView 渲染，支持跨模块自定义事件跳转。
- **多功能背景纸**：内置 Dot/Line/Grid 三种高性能纹理，支持点击触发与数据持久化。
- **基础 Bug 修复**：解决了 Blockquote 样式、Slash 菜单并发事务、DragHandle 漂移等问题。

### 🤖 AI 插件 (AI Plugin)
- **本地 AI 引擎 (Alpha)**：基于 `llama-cpp-python`，支持本地模型加载。
- **硬件环境检测**：自动获取系统内存、CPU 核心数，并提供兼容性拦截。
- **Ollama 深度适配**：支持上下文长度 (num_ctx) 的滑块调节与实时同步。
- **AI 拼写检查 (Spellcheck)**：引入 Aho-Corasick 引擎，支持词库热更新与 120+ 条内置修正规则。
- **像素女仆加载动画**：行内实时 WebP 动画，提供丝滑的 AI 生成反馈。

### 🎨 画板 (Canvas)
- **分组拖拽优化**：移除了 Group 拖拽把手限制，支持分组内节点的批量自由移动。
- **智能右键菜单**：新增「移入/移出分组」功能，支持自动坐标换算与层级管理。
- **Confetti 特效**：集成 canvas-confetti，为特定交互提供庆典反馈。

### 💻 桌面端与后端 (Desktop & Backend)
- **Electron 架构**：基于 Electron-Vite，深度集成 IPC Bridge 通讯协议。
- **本地词库管理**：支持 CSV/文本格式的一键导入，自动生成 Aho-Corasick 自动机。
- **表情包缩略图缓存**：后端自动提取动图首帧作为缩略图，大幅降低前端内存占用。
- **数据库健壮性**：增强了 SQLAlchemy 关联逻辑，修复了 404/500 等路由冲突问题。
