神罗工业化笔记系统

这是一篇模拟笔记。在 Phase 3 中，我们引入了多维主题架构。当前的风格采用了高对比度的暗色背景与霓虹高亮，旨在模拟 FF7 神罗（Shinra）风格的控制台感。

边框和阴影均采用了基于 CSS 变量的动态渲染。通过调整 --accent-color，整个系统的视觉语言可以在瞬间从“科技金属”切换到“古代文明”。

所有的交互组件，包括侧边栏和按钮，都配备了细腻的悬停动画和过渡效果，确保 UI 体验在视觉上充满冲击力且具有沉浸感。
